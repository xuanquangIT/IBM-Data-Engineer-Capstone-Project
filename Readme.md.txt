Capstone Overview: Final project in the IBM Data Engineering Professional Certificate, showcasing comprehensive data engineering skills.

Data Management: Create and query data repositories using relational (MySQL, PostgreSQL, IBM Db2) and NoSQL (MongoDB) databases.

Data Warehousing & Analytics: Design and populate a data warehouse, perform Cube and Rollup operations, and execute ETL processes to build data pipelines.

Big Data & Reporting: Utilize Apache Spark for big data analytics and machine learning predictions, and generate dynamic reports and dashboards with Cognos Analytics.
